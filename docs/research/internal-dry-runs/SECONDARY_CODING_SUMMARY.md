# G0 Internal Dry Run 第二编码总结

状态：Completed on 2026-07-20

证据性质：两个 AI 编码者对合成夹具进行的内部工具校准，不是真人双人研究，不进入 G0 用户样本、价值、理解度或性能指标。

## 方法

第一编码者分别生成报告并按 `GOLD_STANDARD.csv` 审计。第二编码者重新核对 10 条 Finding、全部报告链接、S-09 排除、S-10 解析失败、Provider/Apply 边界和三态语义，再与第一编码结果比较。

第二编码不是盲法：它用于确认计分可重复，不用于估计真实评审者偏差。

## 校准发现

1. `Locator open` 只证明能打开声称的位置，不证明来源是现行、允许或能支持当前语义；
2. Citation precision 必须同时检查 Claim 支持、时间/版本当前性和材料允许状态；
3. 历史文档可以支持一个历史数字，但不能支持“当前目标”这一带时间语义的 Claim；
4. 正确的边界声明不能抵消严重事实错误，DRY-02 必须保持 `rejected`；
5. 内部文档 Dry Run 没有参与者或产品 UI，因此时间、可用性和真实理解度全部为 `NA`。

## 决定

内部计分工具通过正控与负控校准。三项预埋错误全部检出，编码分歧 0.0%，无 P0 材料边界问题。此结果只关闭 G0.5 的内部 Dry Run 出口项；正式 12 人产品证据门保持未完成。
