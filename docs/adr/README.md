# ArcForge Architecture Decision Records

ADR 用于记录影响多个模块、难以轻易撤销的架构决策。

状态：

- `Proposed`：正在评审；
- `Accepted`：已经通过阶段门；
- `Superseded`：被新 ADR 替代；
- `Rejected`：评审后不采用。
